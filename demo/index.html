Demos

These sample applications are not anything you haven't seen before. The cool thing is that these are entirely defined by their UI-model and do not require any Javascript code at all.

  • To Do
  • AddressBook
  • Wine cellar

The data for these demos is stored in your browser local storage (using Backbone.localStorage).

Look at the ui-models for these demos: To Do - AddressBook - Wine cellar

Evolutility.js is not finished yet. Only Chrome is fully supported for now.
