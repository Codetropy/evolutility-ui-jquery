UI models for your Backbone models

... plus a set of model-driven views for it.


Evolutility provides a set of Backbone views which can change form, volume and structure based on a UI-model. For each object (or entity), all the views are defined by a single UI-model containing the definition of UI (layout, fields, model mapping and behaviors).

The UI is defined in some kind of Minimalist Meta-Model for CRUD Applications.
