Backbone.js gives a model for your data. Evolutility.js will give a model for your the UI for this data.

  • ui-model
    A declarative language to describe application UI models.
  • views
    A set of Backbone.js views for one model or a collection of models.
  • actions
    A set of Backbone.js views for actions.
