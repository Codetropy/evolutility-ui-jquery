documentation :: evolutility.js

Backbone.js gives a model to your data. Evolutility.js will give a model to the UI for this data.

  • ui-model
    A declarative language to define the UI of Backbone models.
  • views
    A set of Backbone.js views for one model or a collection of models.
  • controller
    A controller and a toolbar to interact with the views.
