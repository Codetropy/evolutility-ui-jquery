Documentation

Backbone.js gives models to your data. Evolutility.js gives models to the UI which manipulates this data.

  • ui-model
    A declarative language to define the UI for your Backbone models.
  • views
    A set of Backbone views (for a single model or a collection of models) fully described in a UI model.
  • controller
    A controller and a toolbar to manage these views.

Evolutility is using Javascript, HTML5, CSS3, Backbone, Underscore, jQuery, Bootstrap.

The code is available at GitHub under the AGPL V3 license.
