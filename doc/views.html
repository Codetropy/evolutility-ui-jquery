ui-model - views - controller

Views

All views for a particular model are defined by a single ui-model.

2 sets of views:

  • Evol.ViewOne - Views for one model
  • Evol.ViewMany - Views for for a collection of many models

Evol.ViewOne

Backbone views to edit one model (a record at a time)

Edit

All fields

new Evol.ViewOne.Edit(myConfig);

Mini

Only important fields

new Evol.ViewOne.Mini(myConfig);

JSON

JSON representation of the data.

new Evol.ViewOne.JSON(myConfig);

Evol.ViewMany

Backbone views for a collection w/ many records.

list

Tabular view

new Evol.ViewMany.List(myConfig);

cards

Cards view

new Evol.ViewMany.Cards(myConfig);

charts

Charts

new Evol.ViewMany.Charts(myConfig);
