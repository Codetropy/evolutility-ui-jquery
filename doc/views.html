Views

For a ui-model (mapping a Backbone model), Evolutility provides a set of generic Backbone Views for CRUD and more.

3 types of views:

  • Evol.ViewOne
    Views for one model: View, Edit, Mini (quick-edit), JSON...
  • Evol.ViewMany
    Views for a collection of many models: List, Badges, Charts...
  • Evol.ViewAction
    Views for actions on a collection or a model: Filter, Export...
More Views will be added on the future.

 

 

Evol.ViewOne

Backbone views for one model.

View, Edit, Mini (quick-edit), JSON...

 

 

 View

Shows all fields for viewing (read only). Fields are grouped in panels and tabs.

new Evol.ViewOne.Edit(myConfig);

 

 

 

 Edit

Shows all fields for edition. Fields are grouped in panels and tabs.

new Evol.ViewOne.Edit(myConfig);

 

 

 

 Mini (quick-edit)

Only shows important fields (required or showing as a column in grids) in one panel.

new Evol.ViewOne.Mini(myConfig);

 

 

 

 JSON

JSON representation of the data.

new Evol.ViewOne.JSON(myConfig);

 

Other possible views: Wizard...

 

 

Evol.ViewMany

Backbone views for a collection containing many models.

List, Badges, Charts...

 

 

 List

Gives a tabular view of a collection.

new Evol.ViewMany.List(myConfig);

 

 

 

 Badges

Shows records side by side as badges.

new Evol.ViewMany.Badges(myConfig);

 

 

 

 Charts

Draws charts about the collection.

new Evol.ViewMany.Charts(myConfig);

 

 

 

Evol.ViewAction

Backbone Views for actions on a collection or a model.

Filter, Export...

 

 

 Filter

Filter collection.

new Evol.ViewAction.Filter(myConfig);

 

Filter example: "phone number start by 415" or "for categories Finances and Business"... 

 

 

 Export

Export in different data formats.

new Evol.ViewAction.Export(myConfig);

 

Other possible views: Mass Update, Group...

Options

Evol.ViewOne and Evol.ViewMany

cardinality: '1' for Evol.ViewOne.*, 'n' for Evol.ViewMany.*.

viewName: name of the view (the '*' in the Evol.ViewOne.* instanciated). i.e.: "Edit" or "List".

iconsPath: path to icons for model and list of values.

Evol.ViewOne

button_addAnother: Show a "Save and Add Another" button next to "Save".

Evol.ViewMany

pageSize: 20.

pageIndex: 0.

selectable: false.
